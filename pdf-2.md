# **ICDD PDF-2 phase identification manual**
UNLV\
the new version is available on [GitHub](https://github.com/hanc4-git?tab=repositories).

## prerequisites
flash drive

## pdf-2 setup
1. open **SIeve**
2. open raw data (*.raw) as 1D diffraction pattern
3. **Import**

### search filter
1. **Boolean**
2. **Just**
3. **Apply Filter**

### SIeve
1. **Accept Phase**
2. right-click on plot
3. **Open New Window**
4. **Export** (*.csv, *.xy, *.png)

### phase identification
1. select **Phase**
2. right-click on list
3. **Open PDF**
4. **Print**
5. save pdf (*.pdf)
